How to Use:
Create 3 files in the same folder:

index.html (copy the HTML code)

style.css (copy the CSS code)

script.js (copy the JavaScript code)

Open index.html in your web browser.

Features included:

Fully responsive design (works on mobile/tablet/desktop)

7 essential pages (Home, About, Calendar, Faculty, Resources, Gallery, Contact)

Interactive navigation

Working calendar with events

Gallery with filtering

Contact form (with validation)

Faculty/staff directory

Resource links

Announcements section

To customize:

Replace placeholder text with your school information

Change colors in the CSS :root section

Add your own images

Update calendar events in the JavaScript file

Add real faculty/staff information

No external dependencies needed - it uses Font Awesome icons and Google Fonts via CDN.

This website is fully functional and ready to use! You can host it on any web server or even use it locally on your computer.
